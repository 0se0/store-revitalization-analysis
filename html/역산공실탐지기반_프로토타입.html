역산공실탐지기반 — 청년창업 공간 매칭 (2단계 UI 컨셉)

1단계에서 안전점검이 완료된 공실만 추천하는 청년창업 매칭 기능 프로토타입
⚠️ UI 컨셉 목업 고지: 실제 서비스 데이터가 아니라 화면 흐름을 보여주기 위한 목업이다. 매물 목록은 안전등급모델에서 A·B등급(정기 모니터링 이상 확인)으로 분류된 상권을 참고해 구성했다. 실제 서비스 단계에서는 상가정보 API의 실시간 공실 데이터와 연동한다.
